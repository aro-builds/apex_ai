# Workflow Engine

## Purpose

The Workflow Engine acts as the project manager of APEX.

It receives prompts, generates execution plans, and coordinates all services.

## Responsibilities

- Prompt Interpretation
- AI Planning
- Task Scheduling
- Progress Monitoring
- Retry Failed Tasks
- Result Aggregation
