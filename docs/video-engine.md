# Video Editing Engine

## Purpose

Automatically edit videos using prompt-driven instructions.

## Planned Features

- Scene Detection
- Silence Removal
- Auto Cropping
- Subtitle Generation
- Face Tracking
- Zoom Effects
- Color Grading
- Transitions
- Background Music
- Export Presets

## Technologies

- FFmpeg
- OpenCV
- MoviePy
- PySceneDetect
